# Goal：桌面端版本展示与更新提示

## 总目标

桌面端显示当前版本，发现 GitHub 上存在更高正式版本时提示用户下载，并发布 `0.2.0`。

## Sub goal matrix

| Sub goal | 状态 | 完成标准 |
|---|---|---|
| 版本信息 API | 已完成 | 当前版本、最新版本、更新判断、失败降级均有自动化验证 |
| 侧边栏版本与更新提示 | 已完成 | 页面常驻版本；有更新时显示可点击提示 |
| 文档与发布链路 | 已完成 | README/发版说明/CHANGELOG 与新行为一致 |
| 本地验收 | 已完成 | 自动化检查、服务实跑、Windows 打包均通过 |
| 发布 0.2.0 | 待开始 | tag、Actions、正式 Release 和产物均已回查 |

## 重大决策

- 更新提示以 GitHub 最新正式 Release 为源，不引入静默下载和安装。
- 版本 `0.2.0`：新增用户可见能力，按语义化版本提升 MINOR。
